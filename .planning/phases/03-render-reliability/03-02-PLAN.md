# Plan 03-02: GLB Rendering Coverage and Star Tile Boundary Tests

## Objective

Strengthen GLB rendering regression coverage and verify star tile lookup behavior near convexity boundary:
- Extend GLTFUtils tests for fallback behavior (REND-03)
- Verify star tile lookup tests cover boundary conditions (REND-04)

## Context

Building on 03-01 (render infrastructure and shadow policy), this plan addresses GLB and star tile testing.

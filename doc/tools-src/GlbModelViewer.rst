*******
Example
*******

Display the Phobos model created using `convert_to_normalized_glb`

::

    GlbModelViewer -input resources/shapes/phobos_g_148m_spc_obj_0000n00000_v004.glb

Hit `X` to display the body fixed axes and use the arrow keys to rotate the model.
